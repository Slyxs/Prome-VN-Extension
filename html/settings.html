Tint Configuration
Character Tint/Share World Tint With Characters requires Prome to be enabled.
Select the tint preset to use for the Prome VN UI.
World Tint Settings Set the strength of the world blur. Set the brightness of the world. Set the contrast of the world. Makes the world black and white. Set the hue of the world. Inverts the world colors. Saturates the world colors. Makes the world warmer in color.
Character Tint Settings Set the strength of the character blur. Set the brightness of the character. Set the contrast of the character. Makes the character black and white. Set the hue of the character. Inverts the character colors. Saturates the character colors. Makes the character warmer in color.

LLM Analysis Settings

Connect to an OpenAI-compatible endpoint to analyze messages and choose expressions, backgrounds and CGs. Enable a Custom Textbox below to play the segments back, or check the browser console for the raw results.
Fetch Models

Custom Textboxes
Place a textbox image and its matching JSON config (same file name) in the user/images/prome-textboxes folder, then refresh this list.
Refresh
How long to wait, in milliseconds, after a segment finishes displaying before automatically moving to the next one.
How many characters are revealed per second while streaming text.
Reset Position
Preview
Drag the textbox's handle (top-left grip) on-screen to reposition it. Click the textbox (or wait for auto-advance) to move to the next segment.
Example textbox configuration file (name_area/dialogue_area define the safe areas for the speaker name and dialogue text): 
{
  "fonts": [
    { "font_family": "TitleFont", "font_file": "FancyName.ttf" },
    { "font_family": "ReadabilityFont", "font_file": "CleanDialogue.otf" }
  ],
  "name_area": {
    "top": "5%", "left": "8%", "width": "24%", "height": "10%",
    "align": "left", "vertical_align": "middle",
    "styling": {
      "font_family": "TitleFont", "font_size": "26px", "font_color": "#ffcc00",
      "outline_color": "#000000", "outline_thickness": "2px", "shadow": true
    }
  },
  "dialogue_area": {
    "top": "20%", "left": "10%", "width": "80%", "height": "70%",
    "align": "left", "vertical_align": "top", "line_height": "1.4",
    "styling": {
      "font_family": "ReadabilityFont", "font_size": "18px", "font_color": "#ffffff",
      "outline_color": "transparent", "outline_thickness": "0px", "shadow": false
    }
  }
}
Save the image and its config with the same file name (e.g. my_textbox.png + my_textbox.json) inside user/images/prome-textboxes.
